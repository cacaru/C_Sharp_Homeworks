## File_Downloader
 WinForm을 활용해 서버와 클라이언트를 구성    
#### 서버
  - IP와 PORT 번호를 받음
  - 공개할 폴더의 루트를 선택함
  - 어떤 파일을 받는지 log를 기록함

#### 클라이언트
 - IP와 PORT로 접속
 - 파일을 받을 경로를 설정

## MyExplorer
 WinForm을 활용해 파일 탐색기를 만들기
- 상위에 현재 파일 경로를 표기함
- 좌측에 파일 tree를 표시함
- 중앙에 더블 클릭 액션을 통해 폴더에 진입 및 파일을 열어보기가 가능함
